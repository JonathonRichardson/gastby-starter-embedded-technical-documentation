HTFM (Here's the Full Manual) is a template for making static technical documentation sites.  It is intended for use in source code repositories to document the contents of those repositories.

At it's core, HTFM is a Gatsby starter with useful features for basic technical documentation.  This means you get all the speed and simplicity of the Gastby static site generator.
